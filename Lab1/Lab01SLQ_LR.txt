Lab01: Logistic Regression
===========================

Les étudiants doivent compléter le code afin de créer un regresseur logistique multinomial (maximum entropy). 
Ils vont voir quelques méthodes de classement multi-classes comme One-vs-Rest et One-vs-One.

TOOLS:
------
Python, Jupyter, pandas, scikit-learn, numpy, matplotlib

DATASETS:
---------
Iris, bodyPerformance

    I.2. Cost functions
    I.3. Gradients
    I.4. Final product
II. Application and Analsis
    II.1. Binary Ligistic Regression
    II.2. Multi-class Logistic Regression
